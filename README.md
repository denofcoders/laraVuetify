## Installing
- clone
```bash
git clone https://github.com/denofcoders/laraVuetify.git && cd laraVuetify
```
- generate key
```bash
php artisan key:generate
```
- downloading dependencies
```bash
composer install && npm install
```
- create database laraVuetify
- run migrations
```bash
php artisan migrate
```
- start php server
```bash
php artisan serve
```
- run hot reloading
```bash
npm run dev && npm run watch
```
